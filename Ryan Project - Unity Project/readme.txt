This is a personal game/experience created completely by myself in Unity. Assets were modelled and animated in Blender. 

Programming examples in project:
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
- State machines
- New Input System

Art examples in project:
- Light baking using light probs
- Post processing effects
- Animations and Animation state blending
- UI

Theme of the game: 
This game was a personal project where I see a late friend in heaven.
